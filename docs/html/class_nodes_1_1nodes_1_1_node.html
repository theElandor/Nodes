Decode the WAKEUP or START_AT message.

This method offers a quick way to decode the WAKEUP or START_AT message, used for async or sync start. You should always call this method at the beginning of your protocol. Check leader_election_atw_protocol() as an example.

◆ send_RDY()

send_RDY ( self)

Send RDY message to initializer.

This method is used to send RDY message to the initializer, confirming that this node is ready to receive instructions. This method does not increment the total_message counter as this message is part of the initial handshake.

Returns
